addInstance Manual Page

NAME

addInstance - adds an object to a pad.

SYNOPSIS

addInstance (pad,app_instance,access_control_list,callback)

DESCRIPTION

This method is used to add new app_instance object instances to the pad and consequently to the database. The app_instance objects are wrapper objects around instances of the config object.

PARAMETERS

pad

See pad object.

app_instance

See app_instance object.

access_control_list

This an optional array of access_control objects.

Alternatively it can be a single integer value which represents a bitwise combination of this.ACCESS_MODE object. call_back

A callback function that would be invoke when the object is fully initialized. That is it exists in the content list and in the database.

Examples

var config = {
  dim:{},
  transforms:{trslt:{x:0,y:0},rotate:{cx:0,cy:0,angle:0},scale:{x:1,y:1}},
  pen:{color:"blue",width:0,fill:"yellow"}
};            
this.addInstance({config:config},pad);
