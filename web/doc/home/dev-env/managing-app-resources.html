APP Resources

Managing APP Resources

APP resources are the artifacts that make up your APP, this includes Javascript, image, html and css files. Any files that you need for your APP are your resources. All resources reside under the Resources item of an APP.

  2. Adding Resources
  3. Drag & Drop Resources
  4. Deleting Resources
  5. Rename Resource

Creating Resources

Resources can be added either to the Resources item or to folders under the Resources item.

  • Right click Resources item.

    Or right click on any folder under Resources.
  • Select the type of resource you wish to create. Only certain resources can be created specifically html,css,javascript and text files.

Adding Resources

In addition to creating resources, resource can be add by uploading files and zipped folders.

  • Right click Resources item.

    Or right click on any folder under Resources.
  • If a zip is added it will be unzipped and its content made accessible.

Drag & Drop Resources

  • APPs resources, like categories, can be reorganized by dragging and dropping files and folders.

Deleting Resources

  • Right click an APP resources item and select delete.

Rename Resource

Files and folders can be renamed.

  • Right click an APP resources item and select rename.
