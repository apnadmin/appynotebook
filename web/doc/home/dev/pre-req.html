1.1 Skills Prerequisite

Browser-based development.

The main skill set required for building APPs is browser based development. There is no server-side development involved in building APPs. Specifically you need to be comfortable with:

  • JavaScript (including JQuery)
  • SVG (JQuery SVG in particular)
  • CSS/Canvas

Additionally, if you are going to build Sims for Science and Math then it is assumed you'll be familiar with the subject matter and can build correct Sims with the correct animation. There are a number of JavaScript based physics engines available online for physics simulations.
