Imports and Exports

Importing and exporting pages and binders

Users can export their content for external use or printing. Content can be exported to PDF,PNG & SVG.

Similarly users can import PDF, Microsoft word and powerpoint documents.

  1. To import into an existing binder, right-click on the binder and select "Import Office/PDF Document".
  2. To import as a new binder, right-click on your name in the navigation panel and select "Import Office/PDF Document".

    The imported file would have the same name as the file imported.
Note that Microsoft word documents can only be imported as a single page document, regardless of how many pages the document actually has.
